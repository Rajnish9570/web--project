What is hex color code ?
Hex color code is a hexa decimal value of color. It is a way to represent a color in RGB format. It contain only 6 digits with # symbol. After # symbol we write first two digits for red and second two digits for green and last two digits for blue. In hex color code contain number 0 to 225 and alphabet A to F.

What is RGB ?
RGB stands for R- red, G- green, B- blue. computer or any digital divice are understand only three colors that is RGB (red, green, blue). All others colors formed by mixing only this three color (RGB) in different different ratio. Example- 100% red; 100% green; 100% blue = White. The other Example is- 00% red; 00% green; 00% blue = Black.
